PowerShell Tools

Aplikacja desktopowa oparta o PowerShell. Służy do czyszczenia plików tymczasowych, wyszukiwania plików według rozmiaru (zakres MB) oraz pobierania i instalowania wybranych aplikacji. Wszystkie operacje wykonywane są przez skrypty PowerShell. Pliki i skrypty znajdują się w katalogu bin (w tym pliki cn / cncn używane przez aplikację). Projekt działa na systemie Windows
